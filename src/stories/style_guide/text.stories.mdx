# Text classes

Css class for setting text semantic, priority and/or appeareance 

<!-- Don't forget to replace (") with (```) when you copy the snippet to your own app -->

## Header 0 
The highest priority that text can have in the application since is the bigest text and is de only that has gradient background color and has light🌞/dark🌚 mode.

```html
<h1 class="header--0"> The Pokemon </h1>
```
<div class="bg--content pl--1 round">
  <h1 class="header--0"> The Pokemon </h1>
</div>

## Link

Appereance for links, it has light🌞/dark🌚 mode.

```html
<a class="text--link" href="#">Do not open this link 👈</a>
```
<div class="bg--content p--1 round">
  <span class="text--link" href="#">
    Do not open this link 👈
  </span>
</div>

# Background classes

Css classes for backgrounds

```html

<div class="bg--content p--3 round text--content m--1">.bg-content</div>
<div class="bg--lead p--3 round text--lead m--1">.bg-lead</div>

<div class="bg--content p--5 round">
  <div class="bg--primary p--2 round text--content m--3">.bg-primary</div>
  <div class="bg--secondary p--2 round text--content m--3">.bg-secondary</div>
  <div class="bg--highlight p--2 round text--content m--3">.bg-highlight</div>
  <div class="bg--accent p--2 round text--content m--3">.bg-accent</div>
  <div class="bg--danger p--2 round text--content m--3">.bg-danger</div>
</div>

```

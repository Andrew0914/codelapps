# Responsive utilities

The next mixins allow us to apply specific styles on certain pre-defined breakpoints


```css
@include on-mobile() { // <= md
  .my-class{
    width: 100%;
  }
}

@include on-desktop() { // > md
  .my-class{
    width: 480px;
  }
}

```

### Breakpoints
```css
$breakpoints: (
  xs: 0,
  sm: 576px,
  md: 768px,
  lg: 992px,
  xl: 1200px,
  xxl: 1400px,
);
```
